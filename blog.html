Important FAQ

What is Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing scripts (like JavaScript) to dynamically access, modify, and interact with the HTML content of the page. Essentially, it enables live updates and interactivity on web pages.

How do you select an element from the DOM ?

You can select an element from the DOM using methods like `document.getElementById('id')` for selecting by ID, `document.querySelector('.class')` for selecting the first matching class, or `document.getElementsByClassName('class')` for selecting all elements with a specific class.

What is event delegation in the context on DOM,And why is it useful ?

Event delegation is a technique where a single event listener is added to a parent element instead of multiple listeners on individual child elements. This approach is useful because it improves performance by reducing memory usage, simplifies code by minimizing the number of event listeners, and allows handling events for dynamically added elements.

How do you manipulate an elements attributes and styles using the DOM ?

You can manipulate an element's attributes using `element.setAttribute('attrName', 'value')` and retrieve them with `element.getAttribute('attrName')`. To change styles, use `element.style.propertyName = 'value'`, like `element.style.color = 'red'`.
